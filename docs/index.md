# Bem vindo às minhas notas de aula

Este é um repositório pessoal de conhecimento, criado para reunir e compartilhar estudos, descobertas e reflexões ao longo da minha trajetória. Os temas abordados circulam entre Data Science, Engenharia de Software, Arquitetura e Urbanismo, conectando tecnologia, planejamento, cidades e processos de construção do espaço.
Este espaço evolui continuamente, acompanhando meu aprendizado e minhas experiências.
